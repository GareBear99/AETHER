# Creator Quickstart

1. Copy `AddonSDK/Template`.
2. Rename the folder to your add-on name.
3. Edit `addon.json`, `panel.json`, and `ui/layout.json`.
4. Replace art in `assets/`.
5. Add optional preset bank in `presets/presets.json`.
6. Run `python scripts/validate_addon.py <your-addon-folder>`.
7. Zip the add-on or publish it as its own repo.

## Creator business model

AETHER core is intended to be creator-friendly.

You can release add-ons for free or sell them. Example:

- Agartha — $1
- Black Halo — free
- Glass Cathedral — $1

The core goal is to let the platform grow without breaking compatibility.
