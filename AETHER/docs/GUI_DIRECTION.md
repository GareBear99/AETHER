# AETHER GUI Direction — Mystical AAAA Pass

## Goal

The GUI should feel like a sacred device, not a stock plugin shell.

The user should understand the mood before touching a single control.

## Core visual pillars

### 1. Orb-first composition
The center of the plugin is the **Aether Orb**.
It acts as the emotional focal point and visually separates AETHER from the Maid tools.

### 2. Runic framing
A subtle ring of symbols and circular contour lines implies invocation, ritual, and motion without becoming gimmicky.

### 3. Gold + violet hierarchy
- violet = aura / spectral / mystery
- gold = sacred detail / premium highlight / touch points
- deep blue-black = void / depth / cathedral backdrop

### 4. Soft instrument-grade contrast
Panels should glow and recede softly rather than using harsh utility borders.
AETHER should feel like a device that lives in twilight.

## What not to do

- Do not make it look like a generic EDM shimmer plugin
- Do not make it look like another SpaceMaid skin
- Do not flood it with tiny technical controls
- Do not use bright flat neon everywhere

## Desired emotional read

The screenshot should say:

"This thing summons choir atmosphere."

Not:

"This is some random ambience effect."
